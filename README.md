# eurocropsml-transfer-learning

Collected publications and code repositories from the *[PreTrainAppEO](https://www.asg.ed.tum.de/en/lmf/pretrainappeo/) ("Pre-Training Applicability in Earth Observation") research project.*

## Dataset

📓 Reuss, J., Macdonald, J., Becker, S. et al. **The EuroCropsML time series benchmark dataset for few-shot crop type classification in Europe.** _Sci Data 12, 664 (2025)._ https://doi.org/10.1038/s41597-025-04952-7

- 🔗 [eurocropsml](https://github.com/dida-do/eurocropsml)\
  _Benchmark dataset for few-shot crop type classification._
  
## Methods

:notebook: Reuss, J., Macdonald, J., Becker, S. et al. **Benchmarking for practice: Few-shot time-series crop-type classification on the EuroCropsML datase.** _ISPRS Open Journal of Photogrammetry and Remote Sensing 19 (2026)._ https://doi.org/10.1016/j.ophoto.2026.100117.

- 🔗 [eurocropsml-meta-learning](https://github.com/jsreu/EurocropsML-Meta-Learning)\
  _Meta-learning for crop type classification transfer learning._

- 🔗 [eurocropsml-ssl](https://github.com/jsreu/EuroCropsML-SSL)\
  _Self-supervised-learning for crop type classification transfer learning._
